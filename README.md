## Get started

> Development system : <br><br>
> OS : MacOS Big Sur 11.6.5 <br>
> Python v3.9.12 <br>
> Pip v22.0.4

### 1 - Clone project
```text

git clone https://github.com/theoprovost/scoring_client.git <dst_folder>

# SSH

```

### 2 - Install dependencies
```text
cd <dst_folder>
python3.9 -m pip install -r requirements.txt
```
